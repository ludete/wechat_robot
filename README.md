# 微信机器人

## 支持的功能

*   购买token
    *   在数据库中记录这笔购买信息
    *   更新用户的余额
*   提取token
    *   在数据库中记录这笔提取信息
    *   更新用户的余额    
*   查询用户的token余额
*   查询用户的历史记录
    *   历史购买记录
    *   历史提取记录
    *   主动打赏的记录
    *   接收打赏的记录
    *   某段时间内的综合查询记录
*   打赏
    *   支持打赏多人功能
        *   更新打赏人的主动打赏记录
        *   更新接受打赏用户的打赏记录
*   查询指定币种的价格
